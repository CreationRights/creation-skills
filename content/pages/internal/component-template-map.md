# **Creation Skills Internal Page: Component and Template Map**

## **Page Identity**

**Creation Skills**  
**C/Skills.**  
**A Creation Rights Division**

## **Document Type**

Internal build planning document

## **Public Status**

Not a public page

## **Purpose**

This page defines the reusable components, page templates, section patterns, and content modules needed to build the Creation Skills website cleanly.

The goal is to make the site feel consistent, premium, and easy to maintain.

Creation Skills has a large content library. The build should not treat every page as a one-off design.

Reusable components should support:

Core launch pages  
Engagement pages  
Audience pages  
Concept pages  
Resource articles  
Utility pages  
Thank-you pages  
Internal planning content where needed

---

# **1\. Component Principle**

## **Headline**

Build the system once. Reuse it across the site.

## **Body**

Creation Skills pages share a common structure.

Most pages need a hero, clear sections, card grids, process steps, split panels, CTA areas, and footer navigation.

The build should use reusable components so every page feels like part of the same brand system.

## **Core Rule**

Reusable components should make the site feel consistent without making every page look identical.

Secondary CTA style is consistent.

# **14\. Build Rule**

## **Headline**

Use components to protect the brand system.

## **Body**

The Creation Skills site should feel like a coherent consulting brand with reusable patterns, not a collection of disconnected pages.

The component system should make it easy to publish future pages from the master document without redesigning the site each time.

## **Final Rule**

Every new page should be built from an existing template unless there is a strong reason to create a new pattern.
