## **Purpose**

This page defines the recommended URLs, SEO titles, SEO descriptions, and routing notes for the Creation Skills launch site.

The goal is to keep the site clean, searchable, and easy to build.

Every public page should have:

Clean URL  
One clear page title  
Unique SEO title  
Unique SEO description  
Primary CTA  
Secondary CTA  
Correct launch status

---

# **1\. SEO Principle**

## **Headline**

Every page should explain one job clearly.

## **Body**

Creation Skills pages should not compete with each other.

Each page should have a clear purpose:

The homepage introduces the brand.  
What We Do explains the model.  
Engagements explains the offers.  
Creation Rights Readiness explains the platform-preparation bridge.  
Book a C/Skills Review converts serious buyers.  
Utility pages support trust, compliance, and routing.

## **SEO Rule**

One page.  
One purpose.  
One primary search intent.  
One clear CTA.

# **10\. SEO QA Checklist**

## **Before Publishing Any Page**

SEO title is unique.  
SEO description is unique.  
URL is clean.  
H1 appears once.  
Page has one clear purpose.  
Primary CTA is clear.  
Secondary CTA is clear.  
Internal links are relevant.  
No placeholder text remains.  
No internal-only notes are public.  
NILP® mark is correct.  
Creation Skills and Creation Rights roles are not confused.  
Legal review completed where needed.
