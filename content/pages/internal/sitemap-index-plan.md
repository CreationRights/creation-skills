**Creation Skills**  
**C/Skills.**  
**A Creation Rights Division**

## **Document Type**

Internal planning document

## **Public Status**

Not a public page

## **Purpose**

This sitemap organizes the full Creation Skills content library so each page can be added to the master document, grouped correctly, and later marked as launch, phase two, resource, utility, or internal-only.

Creation Skills should not launch every page at once.

The full library gives the brand strategic depth.  
The launch site should remain tight, premium, and easy to understand.
