# **Creation Skills Internal Page: Final Menu Structure**

## **Page Identity**

**Creation Skills**  
**C/Skills.**  
**A Creation Rights Division**

## **Document Type**

Internal navigation planning document

## **Public Status**

Not a public page

## **Purpose**

This page defines the recommended navigation structure for the Creation Skills website.

The menu should make the site feel premium, focused, and easy to understand. The full content library can stay in the master document, but the public navigation should remain simple.

The goal is to help visitors quickly understand:

What Creation Skills does.  
How the consulting engagements work.  
Who Creation Skills helps.  
How Creation Skills connects to Creation Rights.  
How to book a C/Skills Review.

---

# **1\. Navigation Principle**

## **Headline**

Keep the top menu tight.

## **Body**

Creation Skills should not expose the full content library in the main navigation.

A crowded menu will make the brand feel less premium and more like a content archive.

The top navigation should stay focused on the core buying path:

Understand the offer.  
Explore engagements.  
See who it helps.  
Understand Creation Rights readiness.  
Book a review.

## **Rule**

If a page does not help a serious visitor move toward understanding or enquiry, it should not be in the top navigation.
