# **Section 5: What We Do Not Do**

## **Eyebrow**

BOUNDARIES

## **Headline**

Clear boundaries keep the ecosystem clean.

## **Body**

Creation Skills has a defined role.

## **Boundary Cards**

### **Not a Tool Reseller**

We do not position Creation Skills as a reseller for software tools. We help define the company process those tools need to support.

### **Not Legal Counsel**

We help design review points, approval gates, operating files, and record requirements. Legal decisions remain with the client and its advisors.

### **Not a Production Agency**

We do not take over the client’s creative output. We help the client build the system around its own production.

### **Not a Public Course Provider**

Formal education and certification belong with CreativeIP.org. Creation Skills transfers practical capability inside consulting engagements.

### **Not the Platform**

Creation Rights remains the platform for rights, NILP®, consent, approvals, metadata, audit, payouts, and compliance-sensitive media records.

# **Section 6: Partner Use Cases**

## **Eyebrow**

HOW PARTNERS FIT

## **Headline**

Different partners support different moments in the company flow.

## **Use Case Cards**

### **A Legal Partner Supports Rights Review**

Creation Skills defines where review triggers happen. The legal partner advises on the legal position, contract language, usage terms, and client-specific risk.

### **A DAM Partner Supports Asset Structure**

Creation Skills defines the asset lifecycle, metadata needs, approval records, and reuse controls. The DAM partner supports configuration and asset system implementation.

### **An Agent Builder Supports Production Agents**

Creation Skills defines the agent role, operating files, blocked actions, human gates, and owner controls. The agent builder supports technical configuration.

### **A Production Partner Supports Delivery**

Creation Skills defines the production path, approval gates, operating files, and delivery record. The production partner creates, edits, localizes, or packages the media.

### **A Risk Partner Supports Evidence**

Creation Skills defines the record model and review points. The risk partner evaluates controls, evidence, insurance readiness, or audit support.

### **CreativeIP.org Supports Literacy**

Creation Skills defines the company process. CIP supports selected leaders with formal standards literacy where needed.
