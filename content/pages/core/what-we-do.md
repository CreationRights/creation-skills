# **Section 4: The Production Agent Layer**

## **Eyebrow**

PRODUCTION AGENTS

## **Headline**

Company-owned agents should have jobs, owners, limits, and review paths.

## **Body**

Creation Skills helps companies configure production agents around specific jobs inside the media creation process.

These agents are not random chats or loose prompts.

They are governed production support with operating files, approved examples, blocked tasks, token limits, output formats, and human checkpoints.

## **Agent Types**

### **Editing Support Agents**

Support edit notes, rough cut summaries, scene selects, clip suggestions, captions, and post-production handoffs.

### **Localization Agents**

Support translation drafts, subtitle preparation, market versions, language QA prep, cultural review, and dub script drafts.

### **Project Production Agents**

Support asset versions, ad concepts, platform copy, presentation drafts, project variants, and delivery checklists.

### **Product Media Agents**

Support product video outlines, image briefs, PDP copy, marketplace versions, feature breakdowns, and social packages.

### **Release and Packaging Agents**

Support release copy, playlist pitch drafts, merch copy, fan updates, catalog refreshes, and territory versions.

### **Internal Production Agents**

Support meeting notes, task routing, file naming, handoff checklists, vendor notes, status updates, and review prep.
