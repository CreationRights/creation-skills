# **Section 6: Where Production Breaks**

## **Eyebrow**

SYSTEM REPAIR

## **Headline**

Where production breaks, we design the control points.

## **Body**

Production breaks in small places.

Missing files.  
Wrong formats.  
Old briefs.  
Late reviews.  
Unclear owners.  
Missing captions.  
Incomplete asset packages.  
Agent outputs that exceed the task.

Creation Skills designs the rules, agents, checks, and approval paths that route the work back into order before delivery fails.

## **Examples**

### **Missing source file**

Route back to producer.

### **Wrong export format**

Return to production owner.

### **Caption file missing**

Notify social or accessibility lead.

### **Asset package incomplete**

Block delivery checklist.

### **Brief changed after production started**

Flag affected agents and review owners.

### **Agent exceeds task limit**

Pause the agent and notify owner.

### **Rights check incomplete**

Block movement into approval or delivery.
