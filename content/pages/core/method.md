# **Section 7: System Repair**

## **Eyebrow**

SYSTEM REPAIR

## **Headline**

The method is built to find where production breaks.

## **Body**

Production usually breaks in small places: missing files, wrong formats, unclear owners, late reviews, old briefs, incomplete asset packages, unapproved reuse, or agents producing outputs outside their task.

Creation Skills designs control points for those moments.

## **Control Point Examples**

### **Missing source file**

Route back to producer.

### **Wrong export format**

Return to production owner.

### **Caption file missing**

Notify social or accessibility lead.

### **Asset package incomplete**

Block delivery checklist.

### **Brief changed after production started**

Flag affected agents and review owners.

### **Agent exceeds task limit**

Pause the agent and notify owner.

### **Rights check incomplete**

Block movement into approval or delivery.

### **Reuse request unclear**

Route to rights review before production begins.

## **Outcome**

The system catches broken handoffs before work gets lost, published incorrectly, or reused without review.
